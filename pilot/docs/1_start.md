# Экспертное интервью: Процесс продаж (AS-IS)

На основе анализа переговорного процесса между менеджером по продажам (Антон, компания «Карфинанс») и потенциальным клиентом, представляю детальное описание процесса продаж.

---

## Общие характеристики процесса

### Цель процесса
Конвертация входящего обращения клиента в заключённую сделку по подбору и доставке автомобиля из-за рубежа с максимальной маржинальностью при сохранении удовлетворённости клиента.

### Ключевые метрики процесса
- **Конверсия обращений в сделки** (CR)
- **Средний чек сделки**
- **Срок закрытия сделки** (от первого контакта до получения предоплаты)
- **NPS / удовлетворённость клиента**
- **Количество судебных разбирательств** (целевое = 0)

### Начало процесса
Входящее обращение клиента (звонок, заявка с сайта, сообщение в мессенджер).

### Завершение процесса
Получение полной оплаты и передача автомобиля клиенту с подписанием акта приёма-передачи.

### Основной измеримый результат
Подписанный договор + полученная предоплата 100 000 ₽ (на этапе продажи), далее — полная оплата и передача авто.

## Роли в процессе

| Роль | Функция | Когда участвует |
|------|---------|-----------------|
| **Менеджер по продажам** | Ведёт клиента от первого контакта до передачи авто, координирует всех участников | Весь процесс |
| **Клиент** | Инициатор процесса, принимает решения, оплачивает | Весь процесс |
| **Подборщик (агент)** | Осматривает авто на месте, делает диагностику | Этап осмотра |
| **Логист** | Организует доставку и растаможку | Этапы логистики |
| **Бухгалтерия** | Принимает платежи, выдаёт документы | Этапы оплаты |
| **Таможенный брокер** | Оформляет растаможку | Этап растаможки |

## Исключения и вариации

| Ситуация | Действие |
|----------|----------|
| Клиент хочет изменить договор | Договор корректируется по согласованию |
| Авто повреждено при транспортировке | Ремонт за счёт компании или возврат средств |
| Авто не прошло проверку подборщика | Поиск альтернативного варианта |
| Клиент отказался после предоплаты | Возврат предоплаты (условия в договоре) |
| Автовоз попал в ДТП | Полный возврат средств клиенту |

## Ключевые инсайты из диалога

1. **Доверие строится на прозрачности**: менеджер акцентирует «8 лет без судов», «собственные автовозы», «всё по договору».

2. **Гибкость договора — конкурентное преимущество**: готовность корректировать договор снимает ключевое возражение опытного клиента.

3. **Поэтапная оплата снижает риски клиента**: 5 платежей вместо одного — психологически проще.

4. **Критичные параметры нельзя игнорировать**: клиент категорически против вариатора и китайских авто — это «убийцы сделки».

5. **Скорость реакции важна**: визитка отправляется сразу, варианты обещаны «буквально сейчас».
